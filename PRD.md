# Marketer Portfolio Website - PRD

## 1. 목표

채용 담당자/클라이언트가 **30초 안에** 마케터의 핵심 역량을 파악하고, 프리랜서로서 신뢰할 수 있는 전문가로 인식하게 하는 포트폴리오 웹사이트.

## 2. 핵심 원칙

| 원칙 | 근거 |
|------|------|
| 3초 룰 | 진입 3초 만에 "누가, 뭘 하는 사람인가" 파악 |
| 데이터 기반 | "~했습니다" → "전환율 340% 상승 (월 $1.2K → $5.3K)" |
| 스킴 가능 | 스크롤 내리며 핵심만 훑어도 인사이트 전달 |
| 모달 없음 | 클릭 없이 하나의 흐름으로 모든 정보 제공 |

## 3. 기술 스택

| 영역 | 선택 | 이유 |
|------|------|------|
| 마크업 | **HTML5** | 프레임워크 없이 순수 구조 |
| 스타일 | **CSS3 (CSS Variables)** | 커스텀 프로퍼티 활용, 유지보수 용이 |
| 인터랙션 | **Vanilla JS** (필요시) | 가벼운 애니메이션, 숫자 카운트업 |
| 폰트 | Google Fonts (Inter) + system font | 가독성 + 별도 CDN 의존 최소화 |
| 배포 | **GitHub Pages / Netlify** | 정적 사이트 무료 호스팅 |

## 4. 페이지 구조 (Single Page)

### Hero (First Fold - 3초)

```
────────────────────────────────────
  [이름]
  그로스 마케터 | 콘텐츠 전략가      ← 키워드 3개, 직관적 직무

  "퍼널 전환율 340% 개선,
   18개월 만에 신규 유저 12만 유치"  ← 가장 임팩트 있는 숫자 1개

  [프로젝트 보기] [이력서 다운]       ← CTA 2개
────────────────────────────────────
```

### 핵심 숫자 (Impression Strip - 5초)

수평으로 주요 KPI 4개를 일렬로 배치.

| 지표 | 포맷 |
|------|------|
| 총 매출 기여 | "$2.4M+" |
| 캠페인 수 | "47+" |
| 평균 ROI | "340%" |
| 관리 예산 | "$500K+" |

### 경력 요약 (Timeline - 10초)

수직 타임라인, 각 경력은 1줄 문장 + 핵심 성과.

```
2023-현재  A사  │ 그로스 리드
             ↳ 유료 전환율을 12% → 40% 개선
               월 $50K 광고비 운영, ROAS 3.2x 유지

2021-2023  B사  │ 콘텐츠 마케터
             ↳ 유기적 트래픽 6개월 만에 300% 증가
               SEO 블로그 120편 기획·집행
```

### 프로젝트 (지난 3년 간 성과 - 15초)

**총 [프로젝트 수]+개** 진행. 대표 케이스 3개를 카드로 표시.

카드 구조 (문제-해결-성과):

```
┌─────────────────────────────┐
│ 프로젝트명                    │
│                             │
│ 문제  │ 구체적인 문제 상황     │
│ 해결  │ 실행한 전략/ 액션      │
│ 성과  │ 측정 가능한 결과      │
│                             │
│ [자세히 보기] → 모달/외부 링크 │
└─────────────────────────────┘
```

## 5. 디자인 가이드라인

| 요소 | 규칙 |
|------|------|
| 색상 | 단일 액센트 컬러 (#2563EB 계열) + 뉴트럴 |
| 폰트 크기 | Hero 이름 3rem+, 숫자 2rem+, 그 외 1rem |
| 여백 | 섹션 간 최소 80px |
| 모바일 | 320px 브레이크포인트, 모바일 퍼스트 |
| 레이아웃 | CSS Grid + Flexbox |

## 6. 성능 목표

| 지표 | 목표 |
|------|------|
| Bundle Size | < 50KB (전체 HTML + CSS + JS) |
| First Contentful Paint | < 1s |

| Third-party Requests | 0 (폰트 제외) |

## 9. 콘텐츠 준비 Checklist

포트폴리오 작성 전 아래 정보가 필요:

- [ ] 이름, 직함 (키워드 2-3개)
- [ ] 가장 임팩트 있는 숫자 1개 (문구 포함)
- [ ] 핵심 KPI 4개 (지표명 + 수치)
- [ ] 프로젝트 수 (지난 3년 간 완료한 프로젝트 총 수)
- [ ] 경력 3-4개 (회사, 기간, 성과)
- [ ] 프로젝트 3개 (문제, 해결, 성과)
- [ ] 사용 툴/채널/역량 리스트
- [ ] 프로필 사진 (선택)
- [ ] 연락처 (이메일, LinkedIn, Calendly)
